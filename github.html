Git 的核心邏輯

  • 精確回溯： 像遊戲存檔。你可以看到每一行代碼是在哪一天被修改的。
  • 多人協作： 透過「分支」，你可以跟朋友同時修改同一個專案。

對你有什麼幫助？

避險工具： 寫專題時最怕程式跑不動。有了 GitHub，你可以放心實驗，隨時一鍵退回「還能跑」的版本。

快速上手：常用指令

在終端機輸入：

1. 下載專案：

git clone https://github.com/用戶/專案.git

2. 存檔並備註：

git add .
git commit -m "更新功能"

3. 傳上雲端：

git push origin main

你的三階段進化

初階：雲端備份

學會上傳練習題，確保代碼不會因為電腦壞掉而消失。

中階：AI 輔助

啟用 Copilot，學習利用 AI 幫你優化邏輯，減少重複作業。

高階：社群協作

參與別人的開源計畫，這代表你具備真正的團隊開發能力。
