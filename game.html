大帥哥 製作

CodeCombat：語法基礎與邏輯啟蒙

功能與學習目的

  • 真實語言： 擺脫積木，直接學習 Python 或 JavaScript。
  • 邏輯訓練： 訓練變數、條件分支、循環與基礎算法。
  • 即時反饋： 語法錯誤會立刻顯示，降低新手挫折感。

限制與挑戰

遊戲節奏對部分學生可能過慢；高級關卡依賴付費訂閱；對於大型軟體架構的訓練較為不足。

Roblox Studio：創意與社交協作

核心特色與開發能力

  • 3D 空間建模： 學習物理現象、座標系與幾何學應用。
  • 多人連線架構： 理解伺服器與客戶端數據交換邏輯。
  • 全球發布： 學習產品思維、用戶反饋與社群營運。

限制與挑戰

使用 Lua 語言，與主流職涯連結度稍低；平台環境較雜，需注意青少年網路安全隱私。

Unity：邁向職業級的技術巔峰

為什麼要學 Unity？

  • C# 正統程式： 掌握現代軟體開發核心與物件導向架構。
  • 跨領域應用： 支援 VR/AR、工業模擬、影視動畫製作。
  • 職涯競爭力： 全球主流引擎，是接軌大學專業的最佳門票。

限制與挑戰

學習曲線極度陡峭，新手容易受挫；對電腦硬體配備要求高，開發門檻顯著增加。
